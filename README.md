
# Italica – Restaurant Menu Website

## Project Description

Italica is a modern, responsive restaurant website built using React, Vite, and TailwindCSS.
It is designed for an Italian restaurant based in Lebanon and includes:

- A dynamic menu loaded from JSON
- A fully functional shopping cart using React Context
- A floating cart badge visible across all pages
- Order placement with localStorage persistence
- A visual order tracking timeline
- Multiple pages: Home, About, Menu, Cart, Contact, Track Order
- Clean UI built with TailwindCSS and mobile‑friendly navigation
- This project demonstrates component‑based architecture, state management, routing, and frontend UX design.

## Setup Instructions

Follow these steps to run the Italica project locally:

1. **Clone the repository**

```
git clone <your-repo-url>
cd <project-folder>
```

2. **Install dependencies** 

	- Make sure you have Node.js LTS installed. 
	- Then run: 
```
	npm install
``` 


3. **Start the development server**

```
	npm run dev
``` 

This will start Vite’s development server.  Open the URL it provides.
